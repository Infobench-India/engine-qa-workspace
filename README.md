Create Node js APP
npm i -D @nrwl/node
npx nx g @nrwl/node:app engine-qa-api

Creat Nest js App
npm i -D @nrwl/node
npx nx g @nrwl/nest:app engine-qa-server

To Build dengine-qa-sever
npm run build dengine-qa-sever

To Deploy

docker-compose up --build
